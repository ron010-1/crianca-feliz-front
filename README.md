# Projeto React + Vite, Testes Unitários e Storybook

### Obs: É necessário possuir o Node.js versão 20 ou superior na sua máquina! 

* Inicialmente clone o repositório
```
git clone https://github.com/ron010-1/crianca-feliz-front
```
* Instale as dependências
```
npm install
```
* Para executar a aplicação utilize:
```
npm run dev
```
* Para executar os testes utilize:
```
npm run test
```
* Para executar o storybook:
```
npm run storybook
```
